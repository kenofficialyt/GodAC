GodAC is an open source Minecraft anticheat based on GrimAC, designed to support the latest versions of Minecraft.
It currently supports Minecraft versions 1.8–1.21. Geyser players are fully exempt from the anticheat to prevent false positives.

## Downloads

- Latest releases: [GitHub Releases](https://github.com/kenofficialyt/GodAC/releases)

## Requirements

- Java 17 or higher
- Spigot, Paper, or Folia server

## Building

```bash
./gradlew build
```

The built JAR will be in `bukkit/build/libs/`
